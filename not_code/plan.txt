Création d'un serveur
    \_ L'utilisteur donnes un nom au serveur
        \_ Le formulaire de création prends le nom qui est inscrit
            |_ L'utilisteur est marqué comme admin lors de la création
            |_ Le serveur est ajouté a la BDD
            |_ ajouter plus plus tard

Création de compte
    \_ L'utilisteur donnes un pseudonyme
        \_ le pseudonyme est utilisé comme ID dans l'envoi de messages pour diférencier la provenance des utilisteurs
